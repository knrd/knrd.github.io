I will recode assignments from 2nd week quiz regarding linear regression topic. Solutions are valid for one and multiple variables regression.

Hypothesis function

Let $h_\theta (x)$ be hypothesis function for $n$ features: $$h_\theta (x) = \theta_0 + \theta_1 x_1 + \theta_2 x_2 + \theta_3 x_3 + \cdots + \theta_n x_n$$ This function can be definition by matrix multiplication: $$ \begin{align*} h_\theta(x) = \begin{bmatrix}\theta_0 \quad \theta_1 \quad … \quad \theta_n \end{bmatrix} \begin{bmatrix} x_0 \newline x_1 \newline \vdots \newline x_n \end{bmatrix}= \theta^T x \end{align*} $$ As $X$ we can define matrix with $m$ training examples for $n$ features: $$ \begin{align*} X = \begin{bmatrix} x_0^{(1)} x_0^{(2)} … x_0^{(m)} \newline x_1^{(1)} x_1^{(2)} … x_1^{(m)} \newline \vdots \quad \vdots \hspace{0.5em} … \hspace{0.5em} \vdots \newline x_n^{(1)} x_n^{(2)} … x_n^{(m)} \end{bmatrix} \end{align*} $$ Then vector version of hypothesis function will be: $$H_\theta(X) = X\theta$$

Cost function

Let $J (\theta)$ be cost function: $$ J (\theta) = \frac{1}{2m} \sum\limits_{i=1}^{m} (h_\theta(x^{(i)}) - y^{(i)})^2 = \frac{1}{2m} \sum\limits_{i=1}^{m} (\theta^Tx^{(i)} - y^{(i)})^2 = \frac{1}{2m} (X\theta - y)^T (X\theta - y) $$

import numpy as np

def compute_cost(X, y, theta):
    m = y.shape[0]
    partial_diff = np.matmul(X, theta) - y
    return np.matmul(partial_diff.T, partial_diff) / (2 * m)

Gradient Descent for Multiple Variables

To fit $\theta$ parameter we need to minimize $J(\theta)$ function. For each $\theta$ we will be updating its value: $$ \theta_j = \theta_j - \alpha \frac{1}{m} \sum\limits_{i=1}^{m} (h_\theta(x^{(i)}) - y^{(i)}) x_j^{(i)}\qquad \text{for } j = 0..n $$ where $m$ is number of training examples and $n$ is number of features.

import numpy as np

def gradient_descent(X, y, theta, alpha, num_iters):
    m = y.shape[0]
    J_history = np.zeros(num_iters)

    temp_theta = theta.copy()

    for iter in range(num_iters):
        partial_diff = np.matmul(X, theta) - y
        for n in range(theta.shape[0]):
            to_sum = sum(partial_diff * X[:, [n]])
            temp_theta[n][0] = theta[n][0] - alpha * to_sum / m
        theta = temp_theta.copy()
        J_history[iter] = compute_cost(X, y, theta)

    return [theta, J_history]
